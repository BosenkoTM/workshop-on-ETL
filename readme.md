# Проектный практикум по разработке ETL-решений

Добро пожаловать в репозиторий курса «Проектный практикум по разработке ETL-решений». Этот курс направлен на получение практических навыков проектирования, реализации и автоматизации процессов обработки данных (Extract, Transform, Load) с использованием современных инструментов.

## 📋 О курсе

В рамках практикума рассматриваются ключевые этапы жизненного цикла данных: от извлечения из разнородных источников до загрузки в аналитические хранилища. Особое внимание уделяется оркестрации процессов и обеспечению качества данных.

**Используемые технологии:**
*   **ETL-инструменты:** Pentaho Data Integration (PDI / Spoon)
*   **Языки программирования:** Python (Pandas, Dask)
*   **Оркестрация:** Apache Airflow, Docker
*   **СУБД:** PostgreSQL, MySQL
*   **Визуализация и проектирование:** Draw.io
